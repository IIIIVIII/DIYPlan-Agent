# Motion arrow language

Manual generation should convert natural-language actions into a small set of visual operations.

## Operation vocabulary

- `insert_horizontal`: solid arrow from loose part toward receiving slot.
- `drop_vertical`: solid downward arrow for placing a part onto a pin, plate, or hole.
- `thread_rotate`: curved arrow around a screw/foot.
- `tighten_with_tool`: tool icon plus curved rotation arrow.
- `flip_or_turn`: large curved arrow around full assembly.
- `adjust_level`: up/down arrows at feet plus zoom inset for foot rotation.
- `check_level`: spirit level icon on the tabletop.

## Rules

- Each step should have one primary operation.
- If a step repeats around four arms, draw one detailed operation and label `4x`.
- Avoid multiple competing arrows unless the source page shows them.
