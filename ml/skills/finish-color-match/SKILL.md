# Colour / finish matching

When the reference base is **painted** (red, blue, etc.), bare lumber will NOT match. Always add finish materials to the plan.

## Decision tree

1. **Vivid saturated colour** (red pedestal) → primer + semi-gloss latex tinted to sample hex + sample pot first.
2. **Natural wood tone** → stain sample + clear poly (no paint).
3. **Metal column option** → 3 in steel pipe + powder coat at body shop (skip paint on pipe).

## Required material lines (painted)

- Multi-surface primer (qt)
- Colour-matched interior latex semi-gloss (qt) with `store_query` for Home Depot/Lowe's search
- Paint sample pot (verify before full quart)
- 220 grit + tack cloth (between coats)

## Planner rules

- Include `finish` category materials with realistic US store queries.
- Add assumption: "Bring photo or hex code to paint counter for tinting."
- Mention powder-coat pipe as alternative in assumptions or notes.
