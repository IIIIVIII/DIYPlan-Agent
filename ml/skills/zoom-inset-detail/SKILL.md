# Zoom inset detail

Small mechanical interfaces must be shown in an inset, not hidden in the main drawing.

## Circular inset rules

- Use circular zoom for screws, feet, threaded inserts, and tool rotation.
- Use a leader line from the exact connection point to the circle.
- Put the local count next to the circle (`1x`, `4x`).
- Include hardware SKU text inside or beside the inset when visible.
- Show rotation with a curved black arrow.

## Rectangular inset rules

- Use rectangular inset for hidden underside relationships, plate alignment, and stacked discs.
- Use dashed hidden outlines for obscured parts.
- Use a large vertical arrow for "drop in place".
- Use hand/finger icons only for alignment marks or tabs.
