# Diagram Line Art Control

Use when generating SVG instruction pages.

Rules:
- Use white background, black strokes, limited grey fills, and no decorative color unless the source manual uses it.
- Prefer exploded views and single-action pages over crowded multi-action drawings.
- Use thick black arrows for motion and thin leader lines for detail callouts.
- Keep labels sparse: step number, quantity, SKU, and necessary warning/info icons only.
- Do not add marketing text, decorative cards, gradients, or UI labels inside the manual page.
- Keep stroke widths consistent across all pages.
- Ensure thumbnails and full pages use the same SVG source so small previews do not contradict the main manual.
